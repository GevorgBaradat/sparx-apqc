A process model you can open without the tool it was built in

The complete APQC Process Classification Framework, modelled in Sparx EA and published as 1681 interactive diagrams. Click any element to drill into the level below. Hover for the definition held in the model. Leave a question directly on a diagram, save the file, and send it back marked up.
